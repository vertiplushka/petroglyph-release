# Petroglyph

## Разработка цифрового сервиса с элементами искусственного интеллекта для персонализации туристических маршрутов Алтайского края и Республики Алтай

Проект выполнен в рамках написания выпускной квалификационной работы (ВКР) как стартап.

**Автор:** Подольный Владислав Дмитриевич  
**Учебное заведение:** Алтайский Государственный Университет  
**Институт:** Институт гуманитарных наук  
**Кафедра:** Кафедра культурологии и дизайна  
**Курс:** 4 курс, группа 8.209-1

---

## Стек технологий

| Слой | Технология |
|------|-----------|
| Frontend | Next.js 15, React 19, TypeScript |
| Backend | NestJS 10, TypeScript |
| База данных | MySQL + Prisma ORM |
| Инфраструктура | Docker, Docker Compose, Nginx |

## Библиотеки

**Frontend**
- `leaflet` / `react-leaflet` — интерактивные карты
- `@anthropic-ai/sdk` — интеграция с AI (Claude)
- `react-markdown` — рендеринг Markdown
- `lucide-react` — иконки
- `tailwindcss` — стилизация
- `jose` — JWT на стороне клиента
- `clsx` / `tailwind-merge` — утилиты для классов

**Backend**
- `@nestjs/jwt` / `passport-jwt` — JWT-аутентификация
- `@prisma/client` — ORM для работы с базой данных
- `rxjs` — реактивные потоки данных
